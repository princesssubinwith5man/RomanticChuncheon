# 프로젝트 소개
<br/>

> **Romantic Chuncheon**은 재난지원금 사용처 조회 및 지도 표시, 커뮤니티 기능 구현을 목표로 시작된 프로젝트입니다.<br/>
> Android, Java에 대한 전반적인 학습 및 다양한 API의 활용을 목표로 하였습니다.
<br/>
